Set up


Simply open the file named "boxbox" and drag all the content inside to your server file manager (the same place you should have your other content). It will usually be in a file called "public_html" but it might not be so... After that it should be done and ready to use if its in the right place and other files arent interfeering, if your chatpoints webpage isnt in its own file then put it in one so it doesnt interfere with the index.php



Analytics

Setting up your analytics is easy, visit Google Analytics and set up your site there, after you have done that follow the steps below.

Step 1



go to your admin page

Step 2



Click Tracking info then Tracking Code

Step 3



Now copy and paste your information into the analyticstracking.php file that is included in the boxbox files. Save the file and add it to your server file system.

Step 4



Now you should be done, just confirm it is tracking data. Now you have analytics for your website!
